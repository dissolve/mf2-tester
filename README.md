# mf2-tester
bash scripts to test microformats parsers

to run

    pip install mf2py
    composer install
    gem install microformats
    go get -u willnorris.com/go/microformats

    ./initialize.sh
    ./test-all-php.sh
    ./test-all-python.sh
    ./test-all-ruby.sh
    ./test-all-go.sh
    ./package.sh
